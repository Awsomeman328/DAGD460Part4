Web Accessibility Checklist
User Characteristic Accessible design tip
Unable to see Code all images with ALT text
Unable to percieve colors Avoid using color alone to convey information
Unable to use mouse Be sure all website features can be accessed using keyboard
Unable to hear Add captions to multimedia
Prone to having seizures Avoid content that flashes
Easily disractible Keep the design simple

Well the main problem I see is that who ever coded this site didn't spend attention to detail. Over half of the pictures does not have alternative text, some of images being very important because they are links to other pages in the website. Also, something completely obvious but no one ever noticed it, there's no title or heading to the page. There is also on link that repeats another link already present. Lastly, there apears to be a form missing from the search bar in the top right hand corner of the page, whatever they mean by form.

So there are a lot of easy fixes for all of these problems.

    For the images missing alternative texts, just add them into html. Really, it isn't that hard.
  1. For the missing title, just add one in.
